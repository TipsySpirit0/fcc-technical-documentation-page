Programming Data Types

Int

This is used to represent your basic number values

Int x = 3;

It can be seen in all basic programming languages e.g

  • Python
  • Javascript
  • C#
  • C++
  • Java e.t.c.
String

Used to represent sequence of letters

Can also be seen in most programming languages

str x = "I love code";
Bool

Used to represent boolean operations

Boolean operations are you basic true/false operations

bool x = true;
Float

This is used to represent decimal numbers

Also known as floating point numbers

float x = 3.5;
Char

Used to represent single characters(letters)

not seen in mordern and higher level languages

char x = "p";
